📊 Excel Assignment 1 - Data Exploration
🎯 Project Overview

This project focuses on applying essential Excel functions to analyze a product dataset. It covers numerical calculations, conditional logic, category-based analysis, and text extraction techniques commonly used in business reporting and data analysis.

✅ Tasks Performed
🔢 Data Summary Analysis

Used basic statistical functions to gain insights into the dataset:

Calculate the total value of all products using SUM()
Count the number of products using COUNT()
Determine the average product price using AVERAGE()
📈 Price Insights

Analyzed product pricing by identifying:

💲 Lowest product price with MIN()
💰 Highest product price with MAX()
🏷️ Product Price Classification

Created a Price Range column using the IF() function:

High Price → Products priced at $500 or more
Standard Price → Products priced below $500

This helps quickly categorize products based on their pricing level.

🎯 Conditional Calculations

Applied conditional functions to perform targeted analysis:

SUMIF() → Calculate the total price of products in the Electronics category
COUNTIF() → Count products priced below $100

These functions enable category-specific and condition-based reporting.

🔤 Text Extraction from Product ID

Used text functions to extract meaningful information from Product IDs:

📅 LEFT() → Extract Day (first 2 characters)
🗓️ MID() → Extract Month (4th–6th characters)
🌍 RIGHT() → Extract Country Code (last 2 characters)

This demonstrates how structured text data can be transformed into useful attributes for analysis.
